## 📁 Git Rules

### 🔜 깃 커밋 순서

```
git pull` > `git status` > `git add 폴더명.md` and `git add README.md` > `git commit -m "Upload : 폴더명.md 파일 업로드"` > `git push
```



### 🗂 폴더명

```bash
날짜 + 제목(+스터디형식) + 이름
# 0801_자료구조 퀴즈_의진
```



### 📝 Git Commit Message Convention

```
Update : 08.01. 파일 제목 
Upload : 파일제목.md 파일 업로드
Revise : 수정한 내용 설명
Delete : 삭제한 파일 설명
```
