### 📁 Git Rules

##### 🔜 깃 커밋 순서

```
git pull > git status > git add 폴더명.md and git add 파일명 > git commit -m "Upload : ..." > git push
```



##### 🗂 폴더명

```bash
날짜_제목(+스터디형식)_이름
# 0801_자료구조 퀴즈_의진
```



##### 📝 Git Commit Message Convention

```
Upload : 업로드날짜(0801)_파일제목_이름
Update : 업데이트날짜(0801)_파일제목 
Revise : 수정한 내용_설명
Delete : 삭제한 파일_설명
```
